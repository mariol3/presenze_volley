== Presenze Volley

Semplice software per la gestione degli allenamenti di una squadra.
Comprende utenti, allenamenti e interfacce semplici e veloci.

== Installazione
* Clonare il progetto

* <code>bundle</code>

* <code>rake db:migrate</code>

* Per avere iniziali (compreso admin) rake db:seed
